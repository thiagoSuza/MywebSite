About the Studio

A growing independent studio focused on building games with Unity for multiple platforms.

Charlotte Cat Games is an independent game development studio focused on creating games with Unity. The studio works on both experimental prototypes and commercial projects.

Projects include 2D and 3D games for Desktop, Android and WebGL across multiple genres.

The studio also develops online multiplayer using Photon PUN2.
