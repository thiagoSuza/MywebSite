About the Studio

A small studio building games with real development focus

Charlotte Cat Games is an independent studio centered on Unity development, creating projects that range from quick gameplay experiments to more complete and commercial productions.

The studio works with Unity to create 2D and 3D experiences for PC, Android and WebGL. The goal is not only to make games look good, but also to make them feel clear, responsive and enjoyable to play.

Projects include prototypes, arcade concepts, strategy ideas, racing experiments, educational games and online multiplayer systems. Each project is treated as a chance to improve design, structure and player experience.

Charlotte Cat Games also explores connected gameplay features with Photon PUN2, making it possible to build online experiences and multiplayer prototypes with a solid technical base.

Gameplay first Clear mechanics, readable feedback and player focused design.
Technical consistency Organized systems built for iteration, testing and growth.
Platform flexibility Projects designed for desktop, mobile and browser deployment.

Featured Release

A stronger spotlight on the main commercial project

This section gives more weight to the studio main release and helps the portfolio feel more focused and memorable.

Steam Release

Martian Conquer

Martian Conquer is a turn based strategy game with economic management, developed by Charlotte Cat Games and released on Steam.

The player invades and conquers Earth by managing resources, expanding influence and sending cargo ships to support the campaign with strategic decisions.

The project combines progression, economy and tactical planning, making it one of the clearest examples of the studio direction and production scope.
